O que é "pre"?

A extenção do Visual Studio Code permite uma opção de pré-vizualização em direto para ver a página de conteúdo de portais dentro do Visual Studio Code durante a experiência de desenvolvimento.

O que é "h1...h6?

Esta tag representa o "Título" do artigo.

O que é span?

O span é um container generico em linha para conteúdo fraseado, que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo.

O que é div?

O div é o que define uma divisão em um documento HTML.

O que é CSS?

O CSS é o código que você usa para dar estilo à sua página web.

O que é tagw

A tag é As Tags são usadas para informar ao navegador a estrutura do site.

O que é classe?

A classe é uma forma de indentificar os elementos HTML para serem modificados pelo CSS.

O que é id?

O id é um identificador único para modificar qualquer elemento, mas se ele for usado uma vez, não poderá ser usado novamente.

O que é head?

Em HTML, a tag head é usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site.
